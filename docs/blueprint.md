# **App Name**: Trends Tracker

## Core Features:

- Keyword Configuration: Allow users to add or remove keywords to be tracked. Keywords include: "Democrat" vs "Republican", "liberal" vs "conservative", "MAGA" vs "progressive", "Trump" vs "Biden", "election fraud", "voter suppression", "mail-in ballots", "gun control", "2nd amendment", "mass shooting protests", "Black Lives Matter", "Blue Lives Matter", "police reform", "George Floyd protests"
- Trends Chart Display: Display a list of Google Trends charts (5-year view) for each configured keyword.
- Dynamic Update: Automatically update the trends charts on the page when the keyword configuration is modified. The trending data are provided via inline frames sourced directly from Google Trends.
- AI-Powered Trend Explanation Tool: Use AI to provide a tool that will provide short explanations or summaries of why a trend is behaving the way that it is, sourced from news articles and other resources that are retrieved and assessed at run time using the AI's reasoning abilities.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evoking trust and authority, suitable for analytics and trend visualization.
- Background color: Light gray (#ECEFF1), offering a neutral backdrop to emphasize data and charts.
- Accent color: A vibrant orange (#FF9800) is to be used to highlight key elements and interactive components.
- Body and headline font: 'Inter' sans-serif for a clean, modern and readable UI.
- Use simple, clear icons to represent keywords and actions.
- A clean, card-based layout to display the Google Trends charts and keyword configurations.
- Subtle transitions and animations when updating or modifying the list of keywords and their corresponding trends.
